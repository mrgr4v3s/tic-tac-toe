# Tic Tac Toe - PWA APP
![Alt text](src/assets/complete_program.png?raw=true "Game screenshot")

A Angular 8 study project of the legendary Tic Tac Toe game. 
Did exactly as told in the introduction of the Angular Firebase Project Course, found in <https://fireship.io/courses/angular/>

# Purpose of the project

This project have the goal of understanding the basics of Dumb and Smart components, data-binding and the powerful Angular CLI. 

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
